- Flutter (v3.29.0)
- DevTools (v2.42.2)

## Installation

- Install dio library 
```bash
    dart pub add dio
```
- Move to project branch
```bash
    git checkout feature/networking_http
```
- Run project
```bash
  flutter run
```
